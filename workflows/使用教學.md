## 設定教學

### 設定 1: ANTHROPIC_API_KEY（文章生成用）

1. 去 https://console.anthropic.com/settings/keys
2. 建一個 API key
3. 編輯 `~/.claude/mcp.json`，填入 ANTHROPIC_API_KEY

### 設定 2: Google Search Console（排名數據用）

1. 去 https://console.cloud.google.com
2. 建專案 → 啟用「Google Search Console API」
3. 建 Service Account → 下載 JSON key
4. 去 GSC（https://search.google.com/search-console）
5. 設定 → 使用者和權限 → 新增使用者
6. 填入 Service Account 的 email（xxx@yyy.iam.gserviceaccount.com）
7. 把 JSON 放到專案目錄，命名為 `gsc-service-account.json`

### 設定 3: DataForSEO（精確數據，選用）

1. 去 https://dataforseo.com 註冊
2. 儲值 $50
3. 在 `~/.claude/mcp.json` 加入 dataforseo MCP server

### 設定 4: Google Keyword Planner（選用）

1. 有 Google Ads 帳戶
2. 申請 API Developer Token
3. 建 OAuth2 credentials
4. 設定環境變數 GOOGLE_ADS_*

## 每個專案需要的設定檔

每個要用 a7seo 的專案需要一個 `seo-engine.config.json`：

```json
{
  "projectId": "你的專案名",
  "domain": "你的網址",
  "language": "zh-TW",
  "region": "TW",
  "dbPath": "./data/seo-engine.db",
  "publisher": {
    "adapter": "markdown-files",
    "config": { "outputDir": "./content" }
  }
}
```

或用 CLI 自動生成：
```bash
cd 你的專案目錄
node C:\Users\A7\Desktop\a7seo-mcp\dist\cli.js init
```
